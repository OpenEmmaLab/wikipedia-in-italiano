# Chiedi aiuto

*[Voce originale su Wikipedia](https://en.wikipedia.org/wiki/Get_Help)*

**Chiedi aiuto**, conosciuto come **Contatta il supporto** prima dell'aggiornamento Creators Update di Windows 10, è un'interfaccia integrata per comunicare con gli impiegati del servizio clienti Microsoft su Internet. La schermata iniziale chiede all'utente di specificare un prodotto e di spiegare un problema ad esso relativo. Offre inoltre all'utente link a articoli di aiuto online per il supporto aziendale e IT, il supporto e le vendite del Microsoft Store, e un servizio di assistenza per le persone con disabilità.

Una volta inserito un problema, all'utente viene offerta la possibilità di riesaminarlo e selezionare un prodotto dal seguente elenco basato su icone:

* Windows
* Office
* Xbox
* Skype
* OneDrive
* Store
* Excel
* PowerPoint
* Word
* Outlook
* OneNote
* Azure
* Band
* Rewards
* Business, IT & developer
* Dynamics
* Edge
* Exchange server
* HealthVault
* HoloLens
* Minecraft Education
* Mobile devices
* Mouse, keyboard
* MSDN Subscriptions
* MSN
* Outlook.com
* SharePoint server
* SQL Server
* Surface
* Visio
* Visual Studio

Dopo che l'utente ha selezionato una categoria di prodotto, la schermata successiva presenta link per risolvere il problema, informazioni sul dispositivo, un link per telefonare con un tempo di attesa stimato, un programmatore, e una schermata per chattare con un agente tramite messaggio istantaneo.

Precedentemente, come Quick Assist, Chiedi aiuto veniva aggiornato tramite Windows Update, non dal Microsoft Store, nonostante Chiedi aiuto sia costruito sulla Universal Windows Platform. Tuttavia, a partire dall'aggiornamento Windows 10 20H2, Chiedi aiuto può ora essere aggiornato tramite il Microsoft Store.
