# Getent

*[Voce originale su Wikipedia](https://en.wikipedia.org/wiki/Getent)*

**getent** è un comando Unix che aiuta l'utente a ottenere voci in numerosi importanti file di testo chiamati database. Questo include i database passwd e group che memorizzano le informazioni dell'utente – quindi `getent` è un modo comune per cercare i dettagli dell'utente su Unix. Poiché `getent` utilizza lo stesso servizio di nomi del sistema, `getent` mostrerà tutte le informazioni, incluse quelle ottenute da fonti di informazioni di rete come LDAP.

I database che cerca sono: ahosts, ahostsv4, ahostsv6, aliases, ethers (indirizzi Ethernet), group, gshadow, hosts, netgroup, networks, passwd, protocols, rpc, services e shadow.

La sintassi generale è:

```
getent database [key ...]
```

Thorsten Kukuk ha scritto l'utilità *getent* per la GNU C Library.

## Esempi

Recupera l'elenco degli account utente su un sistema Unix (memorizzato in un database chiamato 'passwd'). Questo mostrerà tutti gli account utente, indipendentemente dal tipo di servizio di nomi utilizzato. Ad esempio, se vengono utilizzati sia il servizio di nomi locale che LDAP per gli account utente, i risultati includeranno tutti gli utenti locali e LDAP:

```
$ getent passwd
root:x:0:0:root:/root:/bin/bash
daemon:x:1:1:daemon:/usr/sbin:/bin/sh
bin:x:2:2:bin:/bin:/bin/sh
sys:x:3:3:sys:/dev:/bin/sh
sync:x:4:65534:sync:/bin:/bin/sync
games:x:5:60:games:/usr/games:/bin/sh
man:x:6:12:man:/var/cache/man:/bin/sh
lp:x:7:7:lp:/var/spool/lpd:/bin/sh
mail:x:8:8:mail:/var/mail:/bin/sh
```

Recupera i dettagli per un particolare utente chiamato joe:

```
$ getent passwd joe
joe:x:1000:1000:Joe,,,:/home/joe:/bin/bash
```

Recupera l'elenco degli account di gruppo su un sistema Unix (memorizzato in un database chiamato 'group'):

```
$ getent group
root:x:0:
daemon:x:1:
bin:x:2:
sys:x:3:
adm:x:4:stefan
tty:x:5:
disk:x:6:
lp:x:7:
mail:x:8:
```
