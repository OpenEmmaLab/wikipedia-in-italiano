# GetDataBack

*[Voce originale su Wikipedia](https://en.wikipedia.org/wiki/GetDataBack)*

**GetDataBack** è un software di recupero dati sviluppato da Runtime Software. Può essere utilizzato per recuperare dati da dischi rigidi esterni e interni, schede di memoria, unità USB, ecc. con i file system FAT, ExFAT, NTFS, Ext, HFS+ e APFS, anche se sono necessarie varianti diverse del programma per ogni file system. È richiesta la registrazione del software per recuperare dati con il software.
