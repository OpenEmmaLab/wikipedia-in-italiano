# Getmail

*[Voce originale su Wikipedia](https://en.wikipedia.org/wiki/Getmail)*

**getmail** è un semplice agente di recupero della posta progettato come sostituto di fetchmail, implementato in Python. Può recuperare la posta da server POP3, IMAP4 e Standard Dial-up POP3 Service, con o senza SSL. Supporta cassette di posta semplici e di dominio (multidrop), filtro della posta tramite qualsiasi programma arbitrario, e supporta una vasta gamma di tipi di destinazione della posta, tra cui mboxrd, maildir e agenti arbitrari esterni di consegna della posta. Ha anche una sintassi di configurazione più semplice rispetto a fetchmail, ma supporta meno protocolli di autenticazione. Il software può anche funzionare come agente base di consegna della posta.

Getmail è software libero ed è concesso in licenza sotto la GNU General Public License versione 2. È scritto e mantenuto da Charles Cazabon.

Il software getmail originale richiede Python 2, che non è più supportato. Un fork denominato getmail6, che non proviene dall'autore originale di getmail, fornisce il supporto per Python 3.
